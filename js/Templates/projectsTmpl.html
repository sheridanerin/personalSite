FamGraph - Family Tree Visualization

Worked on a team of four to create a data visualization app that takes in family tree data from family-search.org and outputs beatiful visual representations that are intelligently organized and exhibited. Utilzed the following technologies: ES2015, HTML5, CSS, AngularJS, MongoDB, MongoLab, NodeJS, Express, Mongoose, Passport - FamilySearch Strategy, Gulp, Stylus, UnderscoreJS, and FontAwesome.

Github repo can be found here.
